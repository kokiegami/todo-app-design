タスク一覧表示:

URL: /tasks

HTTPメソッド: GET

機能概要: 全てのタスクを取得し、一覧画面を表示する。

新規タスク登録フォーム表示:

URL: /tasks/new

HTTPメソッド: GET

機能概要: 新規タスク登録用のフォームを表示する。

新規タスク登録処理:

URL: /tasks

HTTPメソッド: POST

機能概要: フォームからの入力データを受け取り、新しいタスクとしてデータベースに保存する。成功後、タスク一覧へリダイレクト。

タスクの完了:

URL: /tasks/{id}/complete

HTTPメソッド: POST (または PUT)

機能概要: 指定されたIDのタスクを完了済みに更新する。成功後、タスク一覧へリダイレクト。

タスクの削除:

URL: /tasks/{id}/delete

HTTPメソッド: POST (または DELETE)

機能概要: 指定されたIDのタスクをデータベースから削除する。成功後、タスク一覧へリダイレクト。
